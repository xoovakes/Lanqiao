# 求和公式

## 等差级数

- $$ 1 + 2 + ... + n = 0.5 * n * (n + 1) $$
- $$ 1^2 + 2^2 + ... + n^2 = (n * (n + 1) * (2 * n + 1)) / 6 $$
- $$ 1^3 + 2^3 + ... + n^3 = (n^2 * (n + 1)^2) / 4 $$

## 几何级数

- $$ 1 + x + x^2 + x^3 + ... + x^n = (x^{n+1} - 1) / (x - 1) $$
- 要是$|x|<1$，即得到无穷递减几何级数
  $$ 1 + x + x^2 + x^3 + ... + x^n = 1 / (x - 1) $$

## 调和级数

- 在编程中O(1)可以忽略.
  $$ 1 + 1/2 + 1/3 + 1/4 + ... + 1/n = ln(n) + O(1) $$
